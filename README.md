# Safelink
Identify if a url is malicious or not.
Contains two models:
- XgboostClassifier - 89% Accuracy
- RandomForestClassifier - 89% Accuracy

## Quickstart
```
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
flask --app server run
```
